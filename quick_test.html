üö® Common Issues & Troubleshooting

If the application doesn't load:

  • Check that servers are running on ports 3000 and 8080
  • Try accessing: http://localhost:8080/public/
  • Check browser console for errors
  • Ensure all files are in the correct public directory

If search doesn't work:

  • Check that the API server is running on port 3000
  • Verify network requests in DevTools Network tab
  • Check console for JavaScript errors

If styles look broken:

  • Ensure styles.css is loading from the correct path
  • Check for CSS syntax errors in DevTools
  • Verify file permissions on public directory
